Промо-сайты Финансового университета

Разработали дизайн-концепциию промо-сайтов Финансового университета при Правительстве РФ на примере олимпиады для 8–11 классов. Среднее значение конверсии из пабликов VK в заявку на сайте — 21%, цена лида — 67 ₽.

Решение позволяет создавать дополнительные промо-сайты без привлечения дизайнеров: для адаптации под другие мероприятия университета достаточно правки текстов и смены некоторых картинок.
